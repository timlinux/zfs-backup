## What sponsorship funds

- Maintenance — keeping the tool working as OpenZFS, syncoid and
  distributions evolve.
- New features — pulling, pushing, restore flows, observability,
  packaging for more distros.
- Documentation — keeping every menu, flag, and workflow documented
  with the same care as the code.
- Security — supply-chain audits, SBOMs, and prompt response to
  upstream advisories.
